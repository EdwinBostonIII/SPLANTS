## System Requirements & Installation

### Time Required

- **Docker Installation:** 10-30 minutes
- **SPLANTS Setup:** 10 minutes
- **Configuration & Testing:** 15-30 minutes
- **Total:** 1-2 hours

### Step 1: Install Docker

#### For Windows Users

1. **Download Docker Desktop:**
   - Go to: https://www.docker.com/products/docker-desktop
   - Click "Download for Windows"
   - File size: ~500MB

2. **Install:**
   - Double-click installer
   - Use default settings
   - Restart computer

3. **Verify:**
   - Look for whale icon in system tray

#### For macOS Users

1. **Download:**
   - Go to: https://www.docker.com/products/docker-desktop
   - Choose "Mac with Intel chip" OR "Mac with Apple chip"

2. **Install:**
   - Open .dmg file
   - Drag Docker to Applications
   - Open Docker from Applications

3. **Verify:**
   - Look for whale icon in menu bar

#### For Linux Users

```bash
# Download and run installation script
curl -fsSL https://get.docker.com -o get-docker.sh
sudo sh get-docker.sh

# Add user to docker group
sudo usermod -aG docker $USER

# Install Docker Compose
sudo apt install docker-compose -y

# Restart computer
```

### Step 2: Get OpenAI API Key

1. **Create Account:**
   - Go to: https://platform.openai.com/signup
   - Sign up and verify email

2. **Add Billing:**

   - Load $20 credit

3. **Create API Key:**
   - Go to: https://platform.openai.com/api-keys
   - Click "Create new secret key"
   - **Copy immediately!** (starts with `sk-proj-`)
   - Save in safe place
